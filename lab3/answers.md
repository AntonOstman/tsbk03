
# Vad blir den förenklade formeln?

Because normal is scaled vector of Rb and Ra we get
Ra x n = (0,0,0)
Rb x n = (0,0,0)

simplified formula is then

j = -(e + 1)*v-rel / (1/ma + 1/mb)
